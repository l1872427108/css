    # 布局单位

    px、%、em、rem、vw、vh

    # px

    像素px时页面布局的基础单位，一个像素表示终端设备所能显示的最小的区域，像素分为 css像素和物理像素。

    • css 像素，为 web 开发者提供。
    • 物理像素：与设备的硬件密度相关，任何设备的物理像素都是固定的。

    # 百分比

    当浏览器的宽度或高度发生变化，通过百分比单位可以使得浏览器中组件的宽和高随着浏览器的变化而变化，从而实现响应式的效果。一般认为子元素的百分比相对于直接父元素.

    1. width --> 父级的width
    2. height --> 父级的height
    3. margin --> 父级的width
    4. padding --> 父级的width
    5. left、top、right、bottom。 --> 父元素的width, height, width, height.
    6. font-size --> 基于继承得到的 font-size
    7. line-height: 基于当前字体的font-size
    8. transform --> 基于自身的 left、top.
    9. border-radius: 相对于自身
    10. background-size: 相对于背景区的宽高。

    # em

    em： 相对于当前对象内文本的字体尺寸。如果没有被设置过font-size, 则相对于浏览器的默认字体尺寸（默认16px）。

    # rem

    rem: CSS3新增的单位，相对于根元素(html元素)的font-size的倍数。利用rem可以实现简单的响应式布局，可以利用 html 元素中的字体大小与屏幕间的比值来设置font-size的值。可以实现党屏幕分辨率变化时让元素也随之变化。

    # vw/vh

    1. vw: 相对于视窗的宽度，视窗宽度时100vw.
    2. vh: 视觉窗口的高度时 100vh。
    3. vmin：vw和vh中较小值。
    4. vmax: vw和vh中的教大值。
