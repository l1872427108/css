    # 三栏布局的实现

    页面中一共三栏。左右两栏宽度固定，中间自适应布局。

    # 利用绝对定位。

    .outer {
    position: relative;
    height: 100px;
    background-color: red;
}
.left {
    position: absolute;
    width: 100px;
    height: 100px;
    
    background-color: tomato;
}
.right {
    position: absolute;
    width: 200px;
    height: 100px;
    right: 0;
    top: 0;
    background-color: gold;
}
.center {
    height: 100px;
    background-color: gray;
    margin-left: 100px;
    margin-right: 200px;
}

    # flex

    .outer {
    display: flex;
    height: 100px;
    background-color: red;
}
.left {
    width: 100px;
    height: 100px;
    background-color: tomato;
}
.right {
    width: 200px;
    height: 100px;
    background-color: gold;
}
.center {
    height: 100px;
    background-color: gray;
    flex: 1;
}

    # 浮动

    .outer {
    height: 100px;
    background-color: red;
}
.left {
    width: 100px;
    height: 100px;
    float: left;
    background-color: tomato;
}
.right {
    float: right;
    width: 200px;
    height: 100px;
    background-color: gold;
}
.center {
    height: 100px;
    background-color: gray;
    margin-left: 100px;
    margin-right: 200px;
}

<div class="outer">
    <div class="left"></div>
    <div class="right"></div>
    <div class="center"></div>
</div>

    # 圣杯布局

    使用 padding. 三栏都是向左浮动。 然后使用 margin-left: -100%; margin-left: -200px; 然后使用 position: relative; left 挤动。

    .outer {
    height: 100px;
    padding-left: 100px;
    padding-right: 200px;
    background-color: red;
}
.left {
    position: relative;
    left: -100px;
    float: left;
    margin-left: -100%;
    width: 100px;
    height: 100px;
    background-color: tomato;
}
.right {
    position: relative;
    left: 200px;
    float: left;
    margin-left: -200px;
    width: 200px;
    height: 100px;
    background-color: gold;
}
.center {
    float: left;
    height: 100px;
    width: 100%;
    background-color: gray;
}

<div class="outer">
    <div class="center"></div>
    <div class="left"></div>
    <div class="right"></div>
</div>

    # 双飞翼布局

    三栏都向左浮动，中间包裹层放在最上面，宽度占满。

    center内容，挤出来 margin-left和margin-right.

    left、right 移动到指定位置。

    .outer {
    height: 100px;
    background-color: red;
}
.left {
    float: left;
    margin-left: -100%;
    width: 100px;
    height: 100px;
    background-color: tomato;
}
.right {
    float: left;
    margin-left: -200px;
    width: 200px;
    height: 100px;
    background-color: gold;
}
.wrap {
    float: left;
    width: 100%;
    height: 100px;
    background: lightgreen;
}
.center {
    margin-left: 100px;
    margin-right: 200px;
    height: 100px;
    background-color: blue;
}

<div class="outer">
    <div class="wrap">
        <div class="center"></div>
    </div>
    <div class="left"></div>
    <div class="right"></div>
</div>
