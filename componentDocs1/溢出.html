# 溢出

当我们给盒子设置了 width / height ，当盒子里内容太多就会发生溢出现象。

#

  1. 盒子，当内容突破 height 限制，就会溢出，使得盒子一团糟。

  2. 一个单词，横向上受限与盒子。就会导致溢出。

# overflow

使用 overflow 来控制溢出。

  • visible 默认值，就上面的情况。
  • scroll 浏览器总会显示滚动条, 即使没有足够多引起溢出的内容.
  • auto 由浏览器决定是否显示滚动条, 桌面浏览器一般仅仅会在有足以引起溢出的内容的时候这么做。
